# PixelChills
A café and bar ordering website built in PHP with a MySQL backend, developed as a university group project.

## What it does
Visitors can browse a menu split into hot/cold drinks and hot/cold food, create an account, sign in, submit feedback, and check the café's location and opening hours. Password resets are handled through email confirmation links.

## My contribution
Led backend development — database interaction for account creation, sign-in records, and feedback submission. Also contributed to front-end pages alongside the rest of the team.

## Stack
PHP · MySQL (mysqli) · HTML · CSS · JavaScript
